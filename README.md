
# Exoplanet Atmospheric Retrieval

This project conducts advanced research into predicting the atmospheric composition of exoplanets by analyzing NASA spectral data. The core objective is to leverage Deep Learning to automate the retrieval process, transforming high-dimensional scientific data into predictable atmospheric traits.


## 🚀 Key Features

- Large-Scale Data Handling: Utilizes a dataset of 20,000 NASA spectral samples.  
- Dimensionality Reduction: Implements advanced preprocessing to compress 15,000 high-dimensional features down to 200 per sample, significantly optimizing model efficiency and training speed.  
- Deep Learning Architecture: Features a custom-built 1D Convolutional Neural Network (CNN) designed specifically for sequential, high-dimensional scientific data.  
- Scalable Versioning: Efficiently managed over 1.5 GB of model and data files using Git Large File Storage (LFS).


## 🛠 Technical Stack

**Language:** Python

**Deep Learning:** TensorFlow / Keras

**Data Processing:** NumPy, Pandas, Scikit-lear

**Model Management:** Git LFS

## Lessons Learned

- Data Acquisition: Loading NASA spectral records containing complex atmospheric signatures.  
- Preprocessing: Applying dimensionality reduction to address the "curse of dimensionality" and prepare the input for the neural network.  
- Model Training: Training a 1D-CNN architecture over a two-year research period to identify patterns in spectral data.  
- Inference: Predicting the specific atmospheric composition based on learned spectral features.

## Results & Research Impact
This research demonstrates the efficacy of using 1D-CNNs for astrophysical data retrieval. By reducing the feature space without losing critical spectral information, the model achieves high-performance atmospheric prediction on a scale that manual analysis cannot match.
